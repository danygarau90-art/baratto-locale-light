# Baratto Locale Light

Versione super leggera per PC meno potenti: Vite + React, niente Next.js, niente Turbopack, niente Tailwind.

## Avvio

```powershell
npm.cmd install
npm.cmd run dev
```

Apri l'indirizzo che appare nel terminale, di solito:

```txt
http://127.0.0.1:5173
```

## Filosofia

- progetto no profit
- zero pagamenti
- zero chat interna
- contatto Telegram esterno
- annunci locali
- privacy minima
- prima demo anche senza Supabase

## Supabase

La demo gira anche senza Supabase. Quando sarà stabile, collegheremo il database usando `.env.local` e lo schema in `supabase/schema.sql`.
